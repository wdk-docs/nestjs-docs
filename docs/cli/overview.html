概观¶

Nest CLI is a command-line interface tool that helps you to to initialize and develop your applications. It helps on many grounds at once, from scaffolding the project to building well-structured apps. The Nest CLI is based on the Angular Devkit package.

Installation¶

Install the CLI using NPM:

$ npm install -g @nestjs/cli

Basic workflow¶

Once the installation process is finished, you should be able to invoke CLI commands directly from your command line through the nest executable. In order to get a list of available functionalities, use the following command:

$ nest --help
